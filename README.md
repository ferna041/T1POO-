## Requisitos Previos

- Java Development Kit (JDK) 8 o superior
- Make (para compilación automatizada)
- Sistema operativo: Windows, Linux o macOS

## Comandos Make

Para cada etapa, los comandos disponibles son:

```bash
$ make        # Compilar
$ make run    # Ejecutar
$ make clean  # Limpiar archivos .class
```
